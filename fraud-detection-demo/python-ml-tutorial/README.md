# Python ML Tutorial

Hands-on Python and ML lessons for software engineers.

## Prerequisites

- **Python 3.10+**

## Setup

1. Create a virtual environment:

   ```bash
   python3 -m venv .venv
   ```

2. Activate it:

   - **Unix/macOS:** `source .venv/bin/activate`
   - **Windows:** `.venv\Scripts\activate`

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Run

With the venv activated, run the first lesson:

```bash
python lesson1.py
```

One-command run after install: `python lesson1.py`. You should see test accuracy printed.

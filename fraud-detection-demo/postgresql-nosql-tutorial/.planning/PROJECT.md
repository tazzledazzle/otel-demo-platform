# postgresql-nosql-tutorial

## What This Is

A tutorial that teaches software engineers PostgreSQL and NoSQL: when to use relational vs document/key-value stores, schema design, queries, and hybrid patterns (e.g. Postgres JSONB + dedicated NoSQL).

## Core Value

Practical guidance so engineers can choose and use PostgreSQL and NoSQL stores effectively in the same system.

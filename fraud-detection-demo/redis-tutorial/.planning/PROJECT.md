# redis-tutorial

## What This Is

A tutorial that teaches software engineers Redis: data structures, caching, pub/sub, and patterns for sessions, rate limiting, and real-time features.

## Core Value

Hands-on Redis so engineers can use it effectively for caching, queues, and real-time workloads.

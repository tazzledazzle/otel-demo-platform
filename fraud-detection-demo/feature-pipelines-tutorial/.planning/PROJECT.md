# feature-pipelines-tutorial

## What This Is

A tutorial that teaches software engineers feature pipelines: designing, building, and operating feature stores and offline/online feature computation for ML systems. Covers versioning, backfills, and serving patterns.

## Core Value

Hands-on understanding of feature pipelines so engineers can implement and maintain feature stores and reproducible feature computation.

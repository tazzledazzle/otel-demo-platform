# rag-pinecone-tutorial

## What This Is

A tutorial that teaches software engineers RAG (Retrieval-Augmented Generation) with a Pinecone-style vector store: ingestion, indexing, retrieval, and integration with LLMs for question-answering and search.

## Core Value

Runnable RAG pipeline so engineers can build search-augmented LLM apps with vector stores (Pinecone or similar).

  <name>Task 2: Runnable LCEL chain script</name>
  <files>01_hello_chain.py</files>
  <action>
Implement one LCEL chain in a single script that runs synchronously and prints LLM output.

1. **Imports** — Use: `ChatPromptTemplate`, `StrOutputParser` from `langchain_core` (prompts and output_parsers); `ChatOllama` from `langchain_ollama`. Do not use langchain_community for Ollama (per research).

2. **Prompt** — ChatPromptTemplate with system + human message; one variable (e.g. "input" or "question") for the user query.

3. **LLM** — ChatOllama(model="llama3.1", temperature=0) or equivalent. If another model is preferred for compatibility, use a well-known tag (e.g. llama3.2, mistral).

4. **Parser** — StrOutputParser() so chain.invoke() returns a plain string (avoid returning AIMessage; research pitfall).

5. **Chain** — Compose with pipe: prompt | llm | StrOutputParser(). Single sync call: result = chain.invoke({"input": "Say hello in one sentence."}) (or equivalent), then print(result).

6. **No** agents, tools, RAG, or multi-step flows. One file, one chain, one invoke. Use invoke() not ainvoke() (sync script).
  </action>

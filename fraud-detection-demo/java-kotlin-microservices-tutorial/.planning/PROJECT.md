# java-kotlin-microservices-tutorial

## What This Is

A tutorial that teaches software engineers Java or Kotlin microservices: service boundaries, APIs, resilience, observability, and deployment. Hands-on with Spring Boot or Ktor and common patterns.

## Core Value

Runnable examples and clear guidance so engineers can design and implement JVM-based microservices (Java or Kotlin).

# spark-tutorial

## What This Is

A tutorial that teaches software engineers Apache Spark: DataFrames, batch and streaming, and deployment (standalone, YARN, or managed). Covers core APIs and production considerations.

## Core Value

Hands-on Spark so engineers can build and run batch and streaming pipelines with DataFrames and Spark SQL.
